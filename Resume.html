Project: Coin counting by AI

โปรเจคนี้เกิดจากการที่ผมอยากนับเหรียญที่ผมออมใส่ไว้ในกระปุกออมสิน ทว่าเหรียญในกระปุกนั้นมีเยอะเกินไป จึงคิดทำ Project นี้ขึ้นมา บวกกับความร้อนวิชาในการลองเชื่อมต่อ Server Line ด้วย5454454 จึงเกิดเป็นโปรเจคนี้ขึ้นมาครับ แสกน QR-Code เพื่อดูsource code ครับ

วิธีการทั้งหมดอธิบายได้โดยดังนี้:

เริ่มแรกเราต้องการโปรแกรมที่จะสามารถพัฒนาสมอง AI ให้เป็นของตัวเองได้ ผมจึงเลือก CiRA CORE

ผมได้รวบรวมรูปภาพเหรียญมากกว่า 30 ภาพเพื่อไม่ให้ AI overfit เกินไป (ตัวอย่างภาพยิ่งเยอะยิ่งดี)

จากนั้นให้เริ่มกด train ได้เลยแต่ผมปรับ Batch-size ของ Generating file เป็น 16 เพื่อผลลัพธ์ที่แม่นยำขึ้น

เมื่อ รอบ Iteration ถึง 50,000 ผมจึงได้นำสมองตัวนั้นออกมาเช็คว่ามีความแม่นยำหรือไม่

ปรากฎว่า AI มีความแม่นยำพอที่จะสามารถแยกประเภทเหรียญออกได้ จึงหยุด train สมอง AI ครับ

จากนั้นเราจะทำการเชื่อม CiRA CORE กับ line โดยใช้ Ngrok ในการเปิดเซิร์ฟเวอร์

เมื่อใส่ Token เรียบร้อยหมดแล้ว เราจะทำการเขียน Code เพื่อให้ CiRA CORE แสดงผลไปยังแชทไลน์

ภาษาที่ใช้เขียนโปรแกรม ผมใช้เป็น JavaScript ในการเขียน ตัวอย่างอยู่ใน QR Code ครับ

เสร็จแล้วเราก็ลองส่งข้อความลงแชทไลน์เพื่อให้ AI ตอบสนองดู

เมื่อเช็คแล้วว่าไม่มีปัญหาใดๆ จึงถือว่าจบภารกิจครับ
